# 💼 Luis Enrique Marcos - Portfolio

Portfolio personal desarrollado para presentar mi experiencia profesional, habilidades técnicas y proyectos personales como desarrollador de software.

Inspirado visualmente en los principios de diseño de **Liquid Glass (iOS)**, con una interfaz moderna, responsiva y enfocada en la experiencia de usuario.

---

## 🚀 Características

* Diseño moderno con efecto Glassmorphism
* Totalmente responsivo
* Modo oscuro
* Sección de perfil profesional
* Experiencia laboral en formato Timeline
* Habilidades técnicas y tecnologías
* Proyectos personales destacados
* Enlaces directos a LinkedIn, GitHub y CV
* Animaciones suaves e interfaz intuitiva

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* Bootstrap Icons

---

## 📂 Secciones

### 👨‍💻 Perfil

Presentación profesional, fotografía, descripción breve y enlaces de contacto.

### 🌎 Idiomas

Visualización de idiomas y nivel de dominio.

### 🧠 Skills

Listado de habilidades técnicas y herramientas utilizadas durante mi experiencia profesional.

### 💻 Tecnologías

Tecnologías y lenguajes de programación con los que he trabajado.

### 🚀 Proyectos

Proyectos personales desarrollados para aprendizaje y práctica continua.

Entre ellos:

* Fallout 4 Companion
* Cat Explorer

### 💼 Experiencia Laboral

Historial profesional presentado mediante una línea de tiempo interactiva.
